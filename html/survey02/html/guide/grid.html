사용시 style 테그 제거하고 사용

main

.main / 상하 패딩값이 없으며 컨텐츠는 모두 메인 섹션안에 들어가있음

section

.contents / .main > .contents / 패딩값 적용

.bg_gray /contents 추가하면 그레이 색상 배경

.section_b / contents 추가하면 하단 그레이 색상 패딩값을 줌 / 주로 섹션 영역 구분으로 사용

.section_con / 섹션안에 들어가는 컨텐츠 / .main > .section > .section_con / 양옆 .25rem 패딩값 적용

button

.btn_wrap / 하단 기본 버튼 그리드 / .main > .btn_wrap / 메인 바로 다음에 적용

.btn_left / 하단 기본 버튼 그리드 / .main > .section > .section_con > .btn_left / 본문안 컨텐츠에 버튼 넣을 때

.btn_fix / 하단 기본 버튼 그리드 / .main > .btn_fix, btn_fixgr(그라데이션 배경) / 메인 바로 다음에 적용
