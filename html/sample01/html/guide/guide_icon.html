icon

- 기본 간격은 4배수 혹은 8배수를 (4, 8, 12, 16...) 를 기준으로 작업합니다.

- 터치 영역은 순수 영역의 16px 사이즈로 영역을 조절합니다. (단, 일부 버튼의 경우 예외 가능)

- 아이콘 영역은 32px을 기준으로 구성합니다.

- 원본은 Adobe illustration 파일1부, Figma 파일 1부를 통해 이중으로 관리합니다.

- 브랜드 2.0 가이드라인은 참고사항으로 두어, 현행화 되어있는 UI 가이드라인에 맞게 기존 아이콘과 어울리게 작업을 진행합니다.
