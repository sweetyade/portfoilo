color

- 컨텐츠의 중요도와 결과에 따라 Primary / System / Grayscale 로 나누어 사용할 수 있도록 합니다.

- 컨텐츠의 용도에 따라 컬러는 수정되거나 추가될 가능성은 있으나, 비슷한 톤을 가진 색상은 가이드를 따르도록 합니다.

- 단, 소셜 로그인 버튼 등의 타 서비스의 컴포넌트를 끌어와 사용하는 경우에는 해당하는 서비스의 가이드를 따르도록 합니다.
